Conteúdos estudados até o momento

  • Algoritmos de Programação Introdução aos conceitos fundamentais de algoritmos e programação. Estudo das principais estruturas de dados e suas aplicações. Desenvolvimento de habilidades em resolução de problemas, implementação de algoritmos e programação modular. Enfoque na construção de algoritmos eficientes e em técnicas básicas de programação, como iteração e recursão.
  • Fundamentos de Sistemas Abordagem dos princípios básicos dos sistemas de computação, incluindo a arquitetura de computadores e o funcionamento de sistemas operacionais. Estudo de como os sistemas computacionais processam informações e gerenciam recursos. Envolvem conceitos como ciclos de processamento, gerenciamento de memória, e sistemas de arquivos.
  • Hardware para Computação Exploração dos componentes físicos dos computadores e dispositivos de hardware. Análise da arquitetura de computadores, dispositivos de entrada e saída, e periféricos. Estudo dos princípios de funcionamento dos processadores, memória, e armazenamento, além das práticas de manutenção e diagnóstico de hardware. Inclui a compreensão e manipulação de sistemas binários, abordando a conversão entre sistemas numéricos binário, decimal e hexadecimal, e como esses conceitos são aplicados na representação e processamento de dados no hardware.
  • Fundamentos Web Introdução aos conceitos essenciais para o desenvolvimento web. Estudo das tecnologias básicas da web, incluindo HTML, CSS e JavaScript. Compreensão da estrutura e funcionamento das páginas web, design responsivo e princípios de usabilidade. Abordagem de boas práticas de desenvolvimento web e integração de front-end com back-end.
  • Web Mobile Desenvolvimento de aplicações web para dispositivos móveis. Estudo das técnicas de design responsivo e adaptação de interfaces para diferentes tamanhos de tela. Introdução ao uso de frameworks e bibliotecas específicas para o desenvolvimento mobile, como Bootstrap e React Native. Compreensão das melhores práticas para otimização de desempenho e usabilidade em dispositivos móveis.
  • Programação de Sistemas Abordagem avançada da programação de sistemas e desenvolvimento de software. Estudo de técnicas e ferramentas para a criação de sistemas complexos e de alto desempenho. Compreensão dos conceitos de programação de baixo nível, como gerenciamento de memória e sistemas de arquivos. Análise de técnicas avançadas de desenvolvimento e depuração de sistemas.
  • Engenharia de Software Caracterização de problemas e algoritmos computacionais. Estudo das principais técnicas de desenvolvimento de algoritmos. Revisão e síntese de técnicas básicas. Estudo e implementação de programação dinâmica, estruturas de dados avançadas, algoritmos gulosos, algoritmos de grafos. Notação Assintótica. Análise e Caracterização de classes de complexidade temporal e espacial de problemas.
